# Corruption Perception & Happiness Analysis

## Project Overview

This applied analytics project investigates the relationship between Corruption Perception Index (CPI) scores and happiness levels across countries using data from 2015–2020.

## Research Question

Is there a statistically significant relationship between CPI scores and happiness levels across countries?

## Dataset

- 792 observations
- Multiple countries
- Period: 2015–2020

## Tools Used

- R
- ggplot2
- dplyr
- Linear Regression

## Skills Demonstrated

- Data Cleaning & Preparation
- Exploratory Data Analysis (EDA)
- Data Visualization
- Statistical Analysis
- Linear Regression

## Key Findings

- A positive relationship was observed between CPI and happiness scores.
- Countries with higher CPI scores generally reported higher levels of happiness.
- Linear regression identified CPI as a significant predictor of happiness.

## Project Outcome

Applied analytical and statistical techniques to real-world data and generated evidence-based insights.
